## Table of Contents

## Lab Description

This is about using a virtual terminal to send a character to the MCU, and then the MCU receives it
and sends it to the LCD to be displayed.

## Prerequisites
### Hardware Requirements
- ATmega32 MCU
### Software Requirements
- Proteus Simulation
- MPLAB X IDE
